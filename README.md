# Maruti-Suzuki-DCF-Valuation
Financial modeling &amp; DCF valuation of Maruti Suzuki India Ltd

## Project Objective
This project focuses on building a complete financial model and performing a Discounted Cash Flow (DCF) valuation of Maruti Suzuki India Ltd.

## What the model includes
- Historical Financial Statement Analysis (Income Statement, Balance Sheet, Cash Flow)
- Ratio Analysis
- Forecasting assumptions
- WACC calculation using CAPM
- Beta estimation via regression
- ROIC & reinvestment-based intrinsic growth
- FCFF projections
- Terminal Value estimation
- Equity value per share calculation
- Market price vs intrinsic value comparison

## Files Included
- Excel valuation model (.xlsx)
- Supporting screenshots / outputs

## Key Learning
The project highlights how assumptions around growth, reinvestment and terminal value significantly affect valuation outcomes.
